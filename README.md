# Sweetshop Playwright Tests

This repository contains automated tests for https://sweetshop.netlify.app/ using Playwright with the Page Object Model (POM) structure.

## Project Structure
- `pages/` contains page object classes.
- `tests/` contains test specifications mapped to manual test cases.

## Requirements
- Node.js
- Playwright

### Installation
```
npm install
```

### Running Tests
```
npm test
```

### Run Tests With UI
```
npm run test:ui
```

### Run Tests With Debugging
```
npm run test:debug
```
